📖 How It Works

  • Public Channels - Anyone can subscribe immediately
  • Private Channels - Require a JWT token to subscribe
  • Channel name prefix determines access type: "private-" requires auth
  • Subscribe to both types in the same WebSocket connection
  • Generate tokens with: php artisan larago:token
  • Open multiple browser tabs and subscribe to the same channel to test broadcasting
